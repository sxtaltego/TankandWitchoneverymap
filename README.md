# TankandWitchoneverymap
Forked l4d2 plugin that spawn tank and witch every chapter (original: https://github.com/pa4H/L4D2-pa4H-Plugins/tree/main/L4D2-Plugins/TankandWitchoneverymap)

Maps with early tank spawn (20-50%):
1. c11m1_greenhouse
2. c4m2_sugarmill_a
3. c5m1_waterfront
4. c13m1_alpinecreek

Maps with late tank spawn (60-80%):
1. c1m1_hotel

Maps with without added tank spawn (scripted tank spawn unchanged):
1. c5m5_bridge
2. c7m1_docks
3. c7m3_port
4. c6m3_port
5. c4m5_milltown_escape
6. c13m2_southpinestream
